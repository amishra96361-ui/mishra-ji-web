# mishra-ji-web

एक सरल वेबसाइट प्रोजेक्ट जिसमें सरकारी नौकरियों की लिस्टिंग शामिल है।

## चलाएँ

1. रिपॉजिटरी में रहें।
2. `python flower.py` चलाएँ।
3. ब्राउज़र में `http://localhost:8000` खोलें।

## नौकरी लिस्टिंग

- `jobs.json` — नौकरी डेटा यहाँ रखा गया है।
- `jobs.js` — नौकरी डेटा को पेज पर दिखाता है और खोज की सुविधा देता है।

## विज्ञापन

- `index.html` में AdSense या अन्य विज्ञापन कोड के लिए placeholder जोड़ा गया है।
- असली Google AdSense के लिए `ca-pub-XXXXXXXXXXXX` और `data-ad-slot` को अपने अकाउंट के साथ बदलें।

## फाइलें

- `index.html` — मुख्य पेज
- `styles.css` — स्टाइल
- `flower.py` — लोकल सर्वर
